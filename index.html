A Sua Missão Tripla

A Secretaria Escolar contratou você para desenvolver 3 pequenos sistemas. Eles rodarão juntos em um único programa:

  1. O Boletim: Aprovar ou reprovar por Nota.
  2. A Catraca: Bloquear alunos com excesso de Faltas.
  3. O Almoxarifado: Adicionar novos materiais ao estoque.

TAREFA 1: O Boletim Eletrônico

1

Capturando a Nota (Entrada)

Lógica: O sistema precisa saber a nota antes de julgar. Usamos o prompt() para perguntar e guardamos numa variável.

Digite a primeira linha do seu sistema no Programiz:

var notaFinal = prompt("Professor, digite a nota final do aluno:");
2

Estrutura de Decisão (If / Else)

Lógica: O cérebro do programa. Se (if) a nota for Maior ou Igual (>=) a 70, aprovado. Senão (else), reprovado.

Na linha de baixo do seu código, adicione a estrutura completa (atenção às chaves { }):

if (notaFinal >= 70) {
    alert("Aprovado! Parabéns.");
} else {
    alert("Reprovado! Precisa estudar mais.");
}

TAREFA 2: A Catraca Inteligente

3

Contagem de Faltas

Lógica: Agora vamos avaliar a frequência. Peça ao inspetor o número de faltas do aluno.

Não apague o que já fez! Pule uma linha e continue construindo o seu programa:

var totalFaltas = prompt("Quantas faltas o aluno possui?");
4

Barrando o Excesso (Sinal de Maior)

Lógica: A regra mudou! Se as faltas forem ESTRITAMENTE MAIORES (>) que 15, o aluno reprova por falta.

Adicione o teste da catraca logo abaixo:

if (totalFaltas > 15) {
    alert("Catraca Bloqueada: Reprovado por Falta!");
} else {
    alert("Acesso Liberado: Frequência regular.");
}

TAREFA 3: O Almoxarifado

5

Criando o Estoque (Array)

Lógica: O colégio guarda muitos materiais. Para não criar 100 variáveis, usamos um Array (Lista) com colchetes [ ].

Mais abaixo, inicie o sistema do Almoxarifado com o estoque base:

var materiais = ["Papel A4", "Caneta Azul"];
Página 2
6

Recebendo Nova Encomenda (Push)

Lógica: Primeiro, perguntamos qual é o item novo. Depois, usamos o poder do .push() para empurrar esse item novo para dentro do Array antigo.

Finalize o seu código com a entrada do novo material:

// Pede o nome do item novo
var itemNovo = prompt("Qual material acabou de chegar?");

// Empurra para dentro da lista
materiais.push(itemNovo);

// Mostra a lista atualizada
alert("Estoque atualizado: " + materiais);
