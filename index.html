Objetivo da Missão

Você é o novo Biólogo Chefe do Zoo Safári. Para manter o parque funcionando sem erros, você vai precisar codificar um único superprograma de gestão contendo 10 passos seguidos.

Digite cada etapa abaixo da outra no Laboratório Virtual. Não apague as fases anteriores!

FASE 1: O Início do Turno

1

Identificação do Biólogo

Lógica: (Variáveis e Entradas) O sistema usa o prompt() para perguntar algo e guarda a resposta numa variável.

No editor de texto, inicie o seu programa perguntando o nome do Biólogo e dando boas-vindas:

// --- PASSO 1: BATE PONTO ---
var biologo = prompt("Bom dia! Qual o seu nome de Biólogo?");
alert("Bem-vindo ao Zoo Central, " + biologo + "!");
2

Estoque de Ração

Lógica: (Números) Variáveis numéricas não usam aspas. Servem para guardar quantidades.

O zoológico começa o dia com 100kg de ração. Crie a variável abaixo:

// --- PASSO 2: ESTOQUE INICIAL ---
var racao = 100;
Página 1

FASE 2: A Fauna

3

Mapeando os Recintos (Array)

Lógica: (Arrays) Para listar os animais sem criar dezenas de variáveis, agrupamos tudo numa Lista (Array) usando colchetes [ ].

Registre os animais que já estão no safári principal:

// --- PASSO 3: RECINTOS ---
var animais = ["Leão", "Zebra"];
4

Chegada de Transporte (Push)

Lógica: (Método Push) Listas podem crescer. O comando .push() empurra um novo dado para o final do seu Array.

Um caminhão chegou com um novo habitante! Adicione a "Girafa" aos recintos e mostre a lista:

// --- PASSO 4: NOVO ANIMAL ---
animais.push("Girafa");
alert("Novos moradores! Animais no parque: " + animais);

FASE 3: Alimentação

5

Consumo de Ração (Matemática)

Lógica: (Processamento) O computador pega o valor da variável de ração, subtrai um número (consumo) e salva o novo resultado.

A hora do almoço dos carnívoros chegou. Desconte 30kg de ração do estoque:

// --- PASSO 5: ALMOÇO ---
racao = racao - 30;
alert("Os animais comeram! Ração restante: " + racao + "kg");
Página 2
6

Checagem de Fome (If/Else)

Lógica: (Decisão Simples) SE a ração for MAIOR que zero, está tudo bem. SENÃO, é alerta de fome.

O sistema do zoológico deve checar automaticamente se ainda temos suprimentos:

// --- PASSO 6: ALERTA DE ESTOQUE ---
if (racao > 0) {
    alert("Tudo sob controle. Temos comida suficiente.");
} else {
    alert("ALERTA VERMELHO! O zoológico está sem ração!");
}

FASE 4: Abertura dos Portões

7

Lotação do Safári (Prompt Numérico)

Lógica: Combinar interação com avaliação. Vamos receber a quantidade de turistas do dia.

Um ônibus escolar acaba de chegar. Pergunte ao segurança quantas pessoas estão querendo entrar:

// --- PASSO 7: BILHETERIA ---
var visitantes = prompt("Quantos visitantes chegaram no ônibus?");
8

Precisamos de mais guias? (If >=)

Lógica: (Comparação) O código decide o plano de ação baseando-se no número digitado acima.

Se a excursão tiver 50 ou mais visitantes, precisamos acionar guias extras:

// --- PASSO 8: CAPACIDADE ---
if (visitantes >= 50) {
    alert("Excursão grande! Acionar Guias e Biólogos extras.");
} else {
    alert("Lotação tranquila. Catracas liberadas.");
}
Página 3

FASE 5: Encerramento

9

Resgate Surpresa

Lógica: (Interação Avançada) Perguntamos o nome do animal resgatado, e jogamos essa variável diretamente para dentro do push().

No fim do dia, a patrulha ambiental trouxe um animal machucado. Adicione ele ao parque!

// --- PASSO 9: CENTRO DE RESGATE ---
var resgate = prompt("Qual animal acabou de chegar do resgate?");
animais.push(resgate);
10

Relatório Final e Execução!

Última linha de código: Resuma todas as variáveis finais num único alerta para encerrar o expediente do Zoo.

Finalize o seu superprograma digitando isto e depois CLIQUE EM RUN (Executar) no Laboratório!

// --- PASSO 10: TELA DE STATUS ---
alert("RELATÓRIO DO ZOO\nBiólogo: " + biologo + "\nRação: " + racao + "kg\nAnimais: " + animais);
