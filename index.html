O Desafio de 30 Passos

Desenvolva o diário de uma Aventura em Muqui!

Modo Desenvolvedor: Não vou lhe dar o código pronto. Vou lhe dar um EXEMPLO, e você terá que DEDUZIR e digitar a resposta correta no Laboratório!

Mantenha o laboratório aberto em outra aba e faça os 30 passos juntos:

Acessar Laboratório Virtual

Fase 1: Chegada a Muqui (Strings)

1

O Destino

A primeira coisa é definir o nome da cidade usando uma String (Texto com aspas).

EXEMPLO: var estado = "Espírito Santo"; AGORA VOCÊ: Crie uma variável chamada 'cidade' e guarde a palavra "Muqui".
2

Turista na Área

Peça ao jogador para digitar o nome dele usando o comando prompt.

EXEMPLO: var pessoa = prompt("Qual o seu nome?"); AGORA VOCÊ: Crie a variável 'turista' e faça um prompt perguntando "Como você se chama, viajante?".
3

O Ponto de Partida

Muqui é famosa pela sua arquitetura histórica e antigos casarões.

EXEMPLO: var praia = "Copacabana"; AGORA VOCÊ: Crie a variável 'atracao' e guarde o texto "Sítio Histórico".
4

Mensagem de Boas-Vindas

O sistema deve cumprimentar o turista usando o alert. Use o sinal + para juntar o texto com a variável!

EXEMPLO: alert("Olá " + pessoa); AGORA VOCÊ: Faça um alert dizendo "Bem-vindo a Muqui, " e junte com a variável turista.
Página 1
5

Registro Silencioso

Envie uma mensagem apenas para os desenvolvedores verem usando o console.log.

EXEMPLO: console.log("Teste de sistema"); AGORA VOCÊ: Faça um console.log com o texto "Turista chegou à cidade de Muqui.".

Fase 2: Economia e Energia (Números)

6

Fazendas de Café

Números não usam aspas! Vamos registrar a colheita de uma fazenda histórica.

EXEMPLO: var dinheiro = 500; AGORA VOCÊ: Crie a variável 'sacasCafe' recebendo o número 100.
7

Disposição para o Passeio

O turista começa o passeio com a energia no máximo.

EXEMPLO: var hp = 100; AGORA VOCÊ: Crie a variável 'energia' recebendo o número 100.
8

Venda de Café

A fazenda vendeu algumas sacas de café hoje. Precisamos atualizar (subtrair) o valor antigo.

EXEMPLO: ouro = ouro - 15; AGORA VOCÊ: Atualize a variável 'sacasCafe' sendo igual a ela mesma menos 25.
9

Mostrando o Estoque

Avise ao fazendeiro como ficou o estoque de café.

EXEMPLO: alert("Sua vida é " + hp); AGORA VOCÊ: Dê um alert avisando "Sacas de café no estoque: " juntando com a variável sacasCafe.
Página 2
10

Pausa para o Café

O turista tomou um café fresquinho da roça. Vamos somar mais energia!

EXEMPLO: mana = mana + 50; AGORA VOCÊ: Atualize a variável 'energia' sendo ela mesma mais 30.

Fase 3: O Boi Pintadinho (Booleanos)

11

Época de Folclore

Valores lógicos ativam e desativam funções (true / false).

EXEMPLO: var chovendo = true; AGORA VOCÊ: Crie a variável 'carnavalChegou' recebendo true (sem aspas!).
12

Preparação

A fantasia ainda não está vestida.

EXEMPLO: var luz = false; AGORA VOCÊ: Crie a variável 'fantasiaPronta' recebendo false.
13

O Boi está na Rua!

Como a fantasia está em false, avise o turista para correr e se preparar.

EXEMPLO: alert("A porta está aberta!"); AGORA VOCÊ: Faça um alert com o texto "Ouvimos os tambores! O Boi Pintadinho está vindo!".
14

Entrando na Festa

O turista vestiu a fantasia. Altere a variável para verdadeiro!

EXEMPLO: luz = true; AGORA VOCÊ: Atualize (não use var/let) a variável 'fantasiaPronta' para true.
Página 3
15

Status da Fantasia (Log)

Registre no console o status da preparação do turista.

EXEMPLO: console.log("Luz: " + luz); AGORA VOCÊ: Dê um console.log dizendo "Fantasia pronta: " + fantasiaPronta.

Fase 4: A Mochila do Turista (Arrays)

16

Equipamento Básico

Para levar vários itens no passeio, criamos um Array (Lista) com colchetes.

EXEMPLO: var bolsa = ["Ouro", "Chave"]; AGORA VOCÊ: Crie a variável 'mochila' sendo uma lista com "Câmera" e "Água".
17

O Lanche da Tarde

Pergunte ao turista qual lanche ele quer comprar na praça.

EXEMPLO: var item = prompt("Qual item?"); AGORA VOCÊ: Crie a variável 'lanche' com um prompt "Qual lanche vai levar para o Morro?".
18

Colocando na Mochila (Push)

O comando .push() empurra um novo item para dentro da nossa lista.

EXEMPLO: bolsa.push(item); AGORA VOCÊ: Use o comando push na variável 'mochila', colocando o 'lanche' lá dentro.
19

Visualizando os Itens

Confirme se tudo está guardado antes de subir a montanha.

EXEMPLO: alert("Bolsa: " + bolsa); AGORA VOCÊ: Dê um alert com "Itens na mochila: " e junte a variável 'mochila'.
Página 4
20

Prevenção

Podemos fazer um push direto escrevendo um texto em vez de variável.

EXEMPLO: bolsa.push("Espada"); AGORA VOCÊ: Dê um push na 'mochila' injetando o texto "Protetor Solar".

Fase 5: Subindo o Morro (If/Else)

21

Rumo ao Cristo

O turista quer ver Muqui de cima (Morro do Fortunato). Peça a altitude atual em metros.

EXEMPLO: var peso = prompt("Quantos Kg?"); AGORA VOCÊ: Crie a var 'altitude' recebendo um prompt "Quantos metros você já subiu?".
22

Avaliando a Trilha (Maior/Menor)

O sistema deve tomar uma decisão com o If/Else baseado na altitude.

EXEMPLO:
if (peso < 50) { alert("Leve!"); }
else { alert("Pesado!"); } AGORA VOCÊ: SE (if) a altitude for MENOR que 400, dê um alert "Continue subindo a trilha!". SENÃO (else), alert "Uau! Que vista maravilhosa de Muqui."
23

O Clima Virou

Vamos perguntar ao turista como está o tempo lá em cima.

EXEMPLO: var frio = prompt("Está frio? (sim/nao)"); AGORA VOCÊ: Crie a var 'chovendo' perguntando num prompt: "Começou a chover? (sim/nao)".
Página 5
24

Descer com Cuidado (Comparação Exata ==)

Dependendo da resposta anterior ("sim" ou "nao"), uma dica de segurança é dada.

EXEMPLO:
if (frio == "sim") { alert("Leve blusa!"); }
else { alert("Vá de regata!"); } AGORA VOCÊ: SE 'chovendo' for IGUAL (==) a "sim", dê alert "Desça devagar, as pedras escorregam!". SENÃO, alert "Tire muitas fotos!".
25

Cansaço Acumulado

A trilha exigiu bastante do corpo do turista. Hora de reduzir a bateria.

EXEMPLO: ouro = ouro - 10; AGORA VOCÊ: Atualize a variável 'energia' sendo ela mesma menos 40.

Fase 6: O Fim de Tarde na Praça

26

Descanso no Centro Histórico

O turista está na pracinha perto do Jardim. Qual a temperatura por lá?

EXEMPLO: var temp = prompt("Qual a temperatura?"); AGORA VOCÊ: Crie a var 'temperatura' perguntando num prompt "Qual a temperatura na praça agora?".
27

Sorvete ou Caldo?

As decisões em Muqui dependem muito se o sol já se pôs e esfriou.

EXEMPLO:
if (temp < 0) { alert("Congelando!"); }
else { alert("Aceitável"); } AGORA VOCÊ: SE 'temperatura' for MENOR que 20, dê alert "O clima pede um caldinho quente!". SENÃO, alert "Perfeito para um sorvete na praça!".
Página 6
28

Preparando o Relatório de Viagem

Para não fazer um "alert" gigante e feio, vamos guardar todos os dados numa String longa.

EXEMPLO: var resumo = "Nome: " + nome + " | HP: " + hp; AGORA VOCÊ: Crie a var 'relatorio' e junte: "Turista: " + turista + " | Cidade: " + cidade + " | Energia Final: " + energia.
29

Inserindo a Mochila no Relatório

Para adicionar mais coisas ao mesmo texto sem apagar o anterior, usamos a própria variável + a novidade!

EXEMPLO: resumo = resumo + " | FIM"; AGORA VOCÊ: Atualize 'relatorio' sendo ele mesmo + " | Mochila: " + mochila.
30

O Grande Álbum de Recordações!

Apenas dispare a mensagem que você preparou. Use o \n se quiser quebrar a linha.

EXEMPLO: alert("== RELATÓRIO ==\n" + resumo); AGORA VOCÊ: Dê um último alert juntando o texto "=== DIÁRIO DE VIAGEM ===\n" com a sua variável 'relatorio'!

Testar a Viagem!
