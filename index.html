Desafio 40 Minutos

Missão Starship: Destino Marte

COMUNICADO: Elon Musk convocou você para a equipe principal de software da SpaceX. Codifique a sequência de lançamento em 40 minutos!

Mantenha o Laboratório Virtual aberto em outra aba e siga o manual:

Acessar Laboratório Virtual

Fase 1: Identificação

1

O Engenheiro Chefe

A primeira coisa é registrar o seu nome no sistema de lançamento usando o prompt.

EXEMPLO: var piloto = prompt("Qual o seu nome?"); AGORA VOCÊ: Crie a variável 'engenheiro' e pergunte "Qual o seu nome, Engenheiro?".

Fase 2: Combustível

2

Enchendo o Tanque

O foguete precisa de números exatos para o combustível (sem aspas).

EXEMPLO: var bateria = 100; AGORA VOCÊ: Crie a variável 'combustivel' recebendo o número 100.
3

Teste de Motores

Fizemos um teste nas turbinas Raptor e gastamos energia. Subtraia do valor atual.

EXEMPLO: bateria = bateria - 10; AGORA VOCÊ: Atualize a variável 'combustivel' sendo ela mesma menos 15.

Fase 3: O Compartimento de Carga

4

Carregamento Básico (Array)

Use uma Lista [ ] para guardar o que vamos levar para o espaço.

EXEMPLO: var itens = ["Água", "Ferramentas"]; AGORA VOCÊ: Crie a variável (Array) 'carga' guardando os textos "Satélite" e "Ração".
Página 1
5

Carro no Espaço (Push)

O Elon pediu para colocarmos um carro dentro do foguete na última hora!

EXEMPLO: itens.push("Mapa"); AGORA VOCÊ: Dê um push na sua variável 'carga' adicionando o texto "Tesla Roadster".

Fase 4: O Controle da Missão

6

Autorização de Voo

Precisamos da permissão do CEO para iniciar a contagem regressiva.

EXEMPLO: var passe = prompt("Deseja continuar? (sim/nao)"); AGORA VOCÊ: Crie a var 'elon' com um prompt perguntando "Elon Musk autorizou o lançamento? (sim/nao)".
7

Go / No-Go (If / Else)

Tome a decisão final. Muito cuidado com as chaves { } e o duplo igual ==.

EXEMPLO:
if (passe == "sim") { alert("Avançando!"); }
else { alert("Parando."); } AGORA VOCÊ: SE 'elon' for igual a "sim", dê um alert "Decolagem autorizada! Rumo a Marte!". SENÃO (else), alert "Lançamento Abortado!".
8

O Relatório Final

Gere o registro final da sua missão agrupando as variáveis criadas.

EXEMPLO: alert("Status: Piloto " + piloto + " | Itens: " + itens); AGORA VOCÊ: Dê um alert juntando o texto "Relatório SpaceX: Engenheiro " + engenheiro + " | Carga: " + carga.

Iniciar Contagem Regressiva!

No seu Laboratório Virtual:

1. Clique no botão azul RUN.

2. Responda "sim" na autorização do Elon Musk.

3. Verifique se o Tesla apareceu no seu Relatório Final!

4. Se funcionou, copie TODO o seu código (Ctrl+A / Ctrl+C) e cole na atividade do PDA para garantir a sua nota e a sua vaga na SpaceX!
