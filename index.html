O Desafio Heróico

Os universos colidiram! Você vai codificar um único superprograma de 10 passos onde o jogador escolhe sua liga, convoca aliados e enfrenta o ataque de um supervilão.

Digite o código passo a passo. Não apague os blocos anteriores, construa o programa de cima para baixo!

FASE 1: O Ponto de Ignição

1

Escolhendo o Universo (Input)

Lógica: O sistema precisa saber de qual lado o jogador está. Usamos prompt para perguntar.

No editor do Laboratório Virtual, inicie o simulador:

// --- PASSO 1: INÍCIO ---
var universo = prompt("Escolha seu lado: digite Marvel ou DC");
2

A Convocação (If / Else)

Lógica: Se (if) a escolha foi Marvel, ganha um herói. Senão (else), ganha outro!

Digite a estrutura lógica abaixo (muita atenção aos parênteses e chaves):

// --- PASSO 2: O HERÓI ---
var heroi = "";

if (universo == "Marvel") {
    heroi = "Homem de Ferro";
} else {
    heroi = "Batman";
}
alert("Comandante selecionado: " + heroi);

FASE 2: Formando a Equipe

3

Carregando a Energia

Lógica: Variáveis numéricas não usam aspas. Elas guardam os pontos de vida da equipe inteira.

Toda equipe começa com a vida no máximo. Configure a energia inicial:

// --- PASSO 3: VIDA INICIAL ---
var energia = 100;
4

Vingadores ou Liga? (Arrays)

Lógica: Usamos Arrays (Listas com [ ]) para agrupar o Herói e um novo Aliado em uma equipe só.

Pule uma linha e adicione o primeiro recruta usando um Prompt e a criação do Array:

// --- PASSO 4: A EQUIPE ---
var aliado = prompt("Digite o nome do seu braço direito:");
var equipe = [heroi, aliado];
5

Reforços de Última Hora (Push)

Lógica: O comando .push() serve para injetar um dado novo diretamente no final do Array.

A batalha será dura, convoque mais um herói surpresa para a lista:

// --- PASSO 5: REFORÇOS ---
var recrutaExtra = prompt("Convoque mais um herói urgente:");
equipe.push(recrutaExtra);
alert("Equipe Final: " + equipe);
Página 2

FASE 3: A Guerra

6

O Estalo do Vilão (Matemática)

Lógica: O computador vai processar uma subtração tirando a vida da variável energia.

Thanos ou Darkseid acabou de usar seu poder máximo! Sua equipe tomou muito dano:

// --- PASSO 6: O ATAQUE ---
energia = energia - 60;
7

Alerta de Dano (Concatenação)

Lógica: Mostrar o resultado juntando texto (String) com a variável atualizada usando o +.

Avise ao jogador como estão as coisas após o impacto colossal:

// --- PASSO 7: STATUS ---
alert("Ataque Crítico! A energia da equipe caiu para: " + energia);
8

Semente dos Deuses (If Extra)

Lógica: Se a energia estiver abaixo de 50, usamos um item de cura para adicionar (+ 30) de vida.

A equipe possui uma cura de emergência! Digite a lógica abaixo:

// --- PASSO 8: CURA ---
if (energia < 50) {
    energia = energia + 30;
    alert("Item de cura usado! Energia subiu para: " + energia);
}
Página 3

FASE 4: O Fim de Jogo

9

O Julgamento Final (If/Else)

Lógica: O teste que define quem ganha. Se a energia aguentou o tranco, vitória. Senão, Game Over.

A fumaça abaixou. Verifique quem sobreviveu à batalha do multiverso:

// --- PASSO 9: O RESULTADO ---
if (energia >= 50) {
    alert("VITÓRIA ÉPICA! O universo está a salvo!");
} else {
    alert("GAME OVER... Os heróis tombaram.");
}
10

Tela de Status & Relatório S.H.I.E.L.D.

Digite a sua última linha de código mostrando todos os dados agrupados num único Alerta:

// --- PASSO 10: TELA FINAL ---
alert("STATUS: Universo " + universo + " | Equipe: " + equipe + " | Energia Restante: " + energia);
