Acessar Laboratório Virtual

Fase 1: Memória e I/O

Inicie seu script declarando as variáveis essenciais. Digite um passo debaixo do outro sem apagar o anterior!

1

A Entrada de Dados (Prompt)

Toda Invasão começa com a identificação. O comando prompt captura uma String digitada por você.
// 1. Entrada de Usuário
var hacker = prompt("Acesso restrito. Identifique-se:");
2

A Energia do Invasor (Number)

O tipo Number não usa aspas. Ele define quanto tempo você aguenta no sistema.
// 2. Variável Numérica
var energia = 100;
3

O Estado da Invasão (Boolean)

O tipo Boolean usa true ou false para controlar o status da missão.
// 3. Variável Lógica
var hackRealizado = false;
Página 1

Fase 2: Matemática e Concatenação

4

Comunicação e Concatenação

Usamos o sinal + junto a uma String para juntar (concatenar) textos e exibir com alert.
// 4. Saída de Dados (Output)
alert("Iniciando protocolo de invasão... Bem-vindo(a), " + hacker);
5

O Firewall (Matemática Pura)

O sinal - faz matemática real subtraindo valor da variável numérica.

Você bateu em uma parede de fogo do sistema. Perca energia!

// 5. Operador Aritmético de Subtração
energia = energia - 35;
alert("Dano de Firewall! Energia restante: " + energia);
6

O Desafio Principal (I/O)

Para tentar hackear, precisamos pedir a senha usando o prompt novamente.
// 6. Preparando para a Decisão
var codigo = prompt("Insira a senha de 3 dígitos (Dica: começa com 00 e termina com 7):");
Página 2

Fase 3: Tomada de Decisão

7

A Encruzilhada (If / Else)

Aqui o computador avalia a condição. Usamos o DUPLO IGUAL (==) para fazer uma pergunta: "É igual a?".

Vamos montar a estrutura base. Preste atenção às chaves { }:

// 7. Operador Relacional e Estrutura Condicional
if (codigo == "007") {
    // Passo 8 virá aqui dentro
} else {
    // Passo 9 virá aqui dentro
}
8

O Caminho do Sucesso (TRUE)

Dentro das chaves do if, atualizamos o Booleano para Verdadeiro.

Preencha a área do Caminho Verdadeiro (antes do else):

    hackRealizado = true;
    alert("Senha aceita! Acesso ao Servidor Central concedido.");
9

O Caminho da Falha (FALSE)

Dentro das chaves do else, punimos o jogador com um choque no sistema (matemática).

Preencha a área do Caminho Falso (depois do else):

    energia = energia - 65;
    alert("SENHA INCORRETA! Choque de Defesa Ativado.");
Página 3

Fase 4: O Relatório Final

10

Saída de Dados Completa

O último comando executa tudo de uma vez. Usamos o \n para quebrar a linha e deixar o texto bonito.

Abaixo do fechamento do `else`, dê o comando final do programa:

// 10. O Resultado Final (Concatenação de Múltiplos Dados)
alert("=== STATUS DA MISSÃO ===\nAgente: " + hacker + "\nEnergia Restante: " + energia + "\nSistema Hackeado: " + hackRealizado);
